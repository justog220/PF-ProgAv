Descripción detallada

Clase capaz de aplicarle el algoritmo del pintor a una Imagen.

A partir de un Pixel inicial, busca los vecinos de este teniendo en cuenta cierta nivel de tolerancia. Habiendo identificado un Pixel como vecino, lo pinta de azul.
