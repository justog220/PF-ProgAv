\doxysubsubsection{Descripción detallada}
Clase encargada de mostrar por pantalla una imagen y responder a eventos. 

Tiene una imagen, la cual es capaz de cargar. Conoce el ancho y alto de la ventana y la escala utilizada. Se asocia con un espacio de trabajo para conocer distintas rutas y archivos. Por otro lado, es capaz de responder ante distintos eventos de mouse y teclado. \begin{DoxyAuthor}{Autor}
