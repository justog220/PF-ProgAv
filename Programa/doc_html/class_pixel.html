Descripción detallada

Clase que modela un pixel de una imagen.

Representa un pixel de los distintos tipos de imagen, indistintamente si es de intensidades, RGB o monocromatico. Tiene valores enteros para rojo, verde y azul. Puede cambiar estos valores y devolverlos en caso de ser necesario.
