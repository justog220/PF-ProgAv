Descripción detallada

Clase capaz de gestionar los diferentes tipos de archivos PNM.

Hereda de GestorDeArchivos la capacidad de leer un archivo y de guardar otro. Tiene un archivo y almacena el identificador del tipo de archivo una vez que fue leido.
